# 03 — 職務表深入 (Duty Tables)

> tCodeDuty 系列是本專案的核心。本頁記錄 NM/HL/PT/ST/TU 的差異、完整中類清單、以及與 plan() 的 OFFICE_MID 對應。

## 五個職務表的關係

| 表 | 葉數 | 用途推測 | 與 NM 關係 |
|---|---|---|---|
| **tCodeDutyNM** | 614 | 全職 Normal — **plan() 用這張** | 基準 |
| tCodeDutyHL | 77 | 精簡/高階版（HighLevel？） | 多數 ⊆ NM，少數獨有（如「投資理財人員」） |
| tCodeDutyPT | 525 | 兼職 Part-Time | 525 全在 NM 內；NM 另有 89 葉不在 PT |
| tCodeDutyST | 219 | 短期/特定版 | 子集 |
| tCodeDutyTU | 199 | 家教 TUtor（數學/理化…科目導向） | 家教專用，與 NM 重疊少 |

> 校正職缺職務小類時用 **NM**。若處理兼職職缺改用 PT，家教職缺用 TU。

## OFFICE_MID 與實際中類對照

plan() 的 `OFFICE_MID` 集合應對應 NM 表這些中類（辦公室/文職性質，正解屬此類時寬鬆保留行政支援）：

```
管理幕僚, 人力資源, 行政後勤／總務,     ← 管理幕僚／人資／行政 大類
財務會計, 稽核審計,                    ← 金融保險／財會／稽核
業務推廣, 客服開發, 國際貿易,          ← 業務／貿易／客服／門市
行銷／廣告, 產品／企劃, 專案管理,       ← 行銷／企劃／專案管理
法務專利, 顧問諮詢,                    ← 法務專利／顧問諮詢
出版翻譯,                            ← 影視傳媒／出版翻譯
新聞媒體                             ← 影視傳媒／出版翻譯
```

> ⚠️ 維護注意：若 plan() 的 OFFICE_MID 用了 NM 表沒有的中類名（拼字/全半形不符），該保護會靜默失效。新版 export 後應驗證 OFFICE_MID 每個元素都存在於 `set(MID.values())`。
